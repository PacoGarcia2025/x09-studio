# SPR-001 - Home do X09 Studio

## Objetivo

Substituir completamente a tela padrão do Next.js pela primeira versão da Home do X09 Studio.

---

## Contexto

O X09 Studio será a central de operações da X09 Technology.

Esta tela é o ponto de entrada do sistema.

Ela deve transmitir organização, profissionalismo e simplicidade.

Não queremos uma landing page.

Queremos uma área de trabalho.

---

## Requisitos

Criar uma tela moderna utilizando apenas:

- Next.js
- React
- TypeScript
- Tailwind CSS

Não instalar bibliotecas externas.

---

## Layout

Tela centralizada.

Fundo escuro.

Visual premium.

Muito espaço em branco.

Tipografia grande.

Responsiva.

## Não utilizar

Não utilizar componentes externos.

Não utilizar animações.

Não utilizar gradientes exagerados.

Não utilizar cards coloridos.

---

## Critérios de aceite

A página deve abrir em localhost:3000.

Não deve existir nenhum elemento padrão do Next.js.

Todo o layout deve ser próprio.

Código limpo.

Componentes simples.

Responsivo.
